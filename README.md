# Target Brazil E-commerce Analysis (SQL)

## Project Overview
This project analyzes Target's operations in Brazil using a dataset of **100,000 orders** across 27 states. The goal was to extract actionable insights regarding seasonality, revenue growth, and logistics efficiency.

## Key Findings
* **Massive Growth:** Identified a **136.98% revenue increase** from 2017 to 2018.
* **Regional Leader:** **SÃ£o Paulo (SP)** is the top-performing state in order volume and shipping speed.
* **Delivery Efficiency:** Orders in **Acre (AC)** are delivered nearly **20 days faster** than estimated.
* **Consumer Behavior:** Most orders are placed in the **Afternoon**, with Credit Cards being the primary payment method.

## Tools Used
* **Google BigQuery:** For data extraction and complex transformations.
* **Standard SQL:** Joins, CTEs, Window Functions, and Date Aggregations.
